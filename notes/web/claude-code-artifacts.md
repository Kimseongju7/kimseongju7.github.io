Claude Code가 작업 진행 상황을 **아티팩트**로 캡처할 수 있게 됐다. 아티팩트는 Claude Code의 작업을 살아 있는 공유 가능한 시각적 웹 페이지 — PR 워크스루, 시스템 설명서, 대시보드, 릴리스 체크리스트 등 — 로 바꿔 주며, 세션이 진행되는 동안 스스로 업데이트된다.

## 무엇이 달라지나

Claude Code 세션은 장애 조사부터 서비스 리팩토링, 수개월치 데이터 분석까지 다양하다. 아티팩트는 그 작업을 누구나 열어 탐색할 수 있는 웹 페이지로 번역한다: 풀 리퀘스트 워크스루, 필터·정렬 가능한 대시보드, 작업이 진행되면서 스스로 채워지는 릴리스 체크리스트 같은 것들이다. 공유 작업에 대한 협업이 쉬워져, 팀이 상태 공유에 쓰는 시간을 줄이고 만드는 데 더 시간을 쓸 수 있다.

## 세션 컨텍스트 기반으로 생성

Claude Code는 코드베이스, 커넥터, 대화 자체를 포함한 세션의 전체 컨텍스트로 아티팩트를 만든다. 예를 들어 장애 페이지 하나에 코드의 실패한 테스트와 그 배후 함수, 연결된 모니터링 도구의 에러 급증 그래프, 방금 실행한 세션의 근본 원인 추론을 한데 모을 수 있다. 데이터 소스를 연결하거나 인프라를 세울 필요 없이, 페이지를 요청하면 이미 존재하는 것들로부터 Claude Code가 만들어 준다.

## 제자리에서 갱신되는 라이브 페이지

Claude Code가 아티팩트를 업데이트하면 열려 있는 페이지가 그 자리에서 새로고침되고, 게시되는 순간 팀원들에게 보인다. 모든 게시는 같은 링크의 새 버전이 되며, 버전 히스토리로 언제든 복원할 수 있고, 갤러리에서 자신이 만든 모든 아티팩트를 둘러보고 관리할 수 있다.

내부 테스트에서 가장 흔한 사용 사례는 디버깅이었다. 엔지니어가 스탠드업 전에 장애 조사를 시작하면 Claude Code가 로그를 훑고 타임라인, 의심 커밋, 에러율 차트가 담긴 아티팩트를 게시한다. 페이지 헤더에서 팀에 링크를 공유하고, 스탠드업이 시작될 때쯤이면 조사 진행에 따라 두 번 더 재게시되어 있다. 모두가 같은 컨텍스트의 같은 화면을 보므로 "에이전트가 찾은 걸 설명해 달라"는 과정이 필요 없어진다.

## 조직 내 비공개

모든 아티팩트는 기본적으로 작성자에게만 공개된다. 준비되면 페이지에서 직접 팀원과 조직에 공유할 수 있다. 아티팩트는 조직의 인증된 멤버만 볼 수 있고 외부 공개는 불가능하다. 관리자는 조직 수준 토글과 역할 기반 스코핑으로 접근을 관리하고, 보존 정책을 설정하며, 컴플라이언스 API로 조직 전체 가시성을 확보한다.

## 시작하기 — 역할별 활용 아이디어

세션에 아티팩트를 요청하거나, 그냥 시각적인 것을 요청하면 된다.

- **법무/오픈소스**: 저장소에서 바로 모든 서드파티 의존성과 라이선스를 나열하고 카피레프트를 플래그하는 라이선스 감사
- **프라이버시**: 코드 전반에서 개인정보가 수집·저장·로깅되는 위치를 추적한 데이터 흐름 지도
- **보안**: 정확한 코드 라인에 링크되어 수정이 모호하지 않은 인증(auth) 리뷰 결과
- **FinOps/플랫폼 재무**: Terraform 등 IaC에서 클라우드 리소스와 비용 동인을 서비스별로 매핑
- **소프트웨어 엔지니어**: diff와 주변 코드에서 뽑아낸, 리뷰어가 실제로 따라갈 수 있는 PR/버그 워크스루
- **디자이너·프론트엔드**: 실제 [[컴포넌트]] 라이브러리로 만든 화면 UX 대안 5종(선택하면 바로 배포 가능한 수준)
- **스태프 엔지니어·아키텍트**: 화이트보드가 아닌 실제 임포트 그래프로 그린 서비스 구조 지도
- **SRE·온콜**: 조사하며 자라나 포스트모템이 되는 장애 페이지(타임라인, 의심 커밋, 모니터링의 에러 급증)
- **엔지니어링 매니저**: 머지된 PR로부터 만든 이번 주 팀 출시 내역 페이지(프로젝트별 그룹핑)

Claude Code가 페이지를 만들어 링크를 주면, 브라우저나 데스크톱 앱에서 열어 헤더에서 공유하면 된다. 업데이트는 같은 URL에 자동 게시된다.

## 제공 범위

아티팩트는 Claude Team 및 Enterprise 조직 대상 베타로, Claude Code CLI와 데스크톱 앱에서 사용 가능하며 페이지는 어떤 브라우저에서든 볼 수 있다.
