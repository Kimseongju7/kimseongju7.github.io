## 한 줄 정의

열지도(heatmap)는 행렬 형태의 데이터에서 각 셀의 값 크기를 색의 진하기(또는 색상)로 표현해, 수치의 분포와 패턴을 한눈에 파악하게 해주는 시각화 기법이다.

## 자세히

숫자가 빽빽하게 들어찬 표는 사람이 눈으로 훑어도 패턴을 찾기 어렵다. 열지도는 그 숫자를 색으로 바꿔서 보여준다. 값이 클수록 진한 색(예: 빨강), 작을수록 옅은 색(예: 파랑)을 칠하면, 어디에 큰 값이 몰려 있고 어디가 비어 있는지가 즉시 드러난다. "열(熱)지도"라는 이름도 온도 분포도처럼 뜨거운 곳과 차가운 곳을 색으로 구분하는 데서 왔다.

머신러닝에서는 훈련 전 데이터 탐색(EDA, 탐색적 데이터 분석) 단계에서 자주 쓴다. 대표적인 용도가 **[[상관관계행렬|상관관계 행렬]] 시각화**다. 피처(변수)들 사이의 상관계수를 행렬로 계산한 뒤 열지도로 그리면, 어떤 피처끼리 강하게 연관되어 있는지(다중공선성 위험), 타깃 변수와 상관이 높은 피처는 무엇인지 색만 보고도 판단할 수 있다. 이 밖에 결측치 분포 확인, [[혼돈행렬|혼동 행렬(confusion matrix)]] 시각화, 시간대×요일별 사용량 분석 등 "두 축 × 값" 구조라면 어디든 쓰인다.

머신러닝 밖에서도 널리 쓰인다. 웹/UX 분석에서는 사용자가 페이지의 어디를 클릭하고 어디까지 스크롤했는지를 화면 위에 색으로 겹쳐 보여주는 클릭·스크롤 히트맵(Hotjar, Microsoft Clarity 등)이 있고, 지도 위에 인구 밀도나 주문량을 색 농도로 칠하는 지리 히트맵도 흔하다. GitHub 프로필의 잔디(contribution graph)도 날짜별 커밋 수를 초록 농도로 표현한 열지도의 한 예다.

주의할 점은 색 배색(컬러맵) 선택이다. 상관계수처럼 음수~양수를 오가는 값은 0을 중심으로 양쪽으로 갈라지는 발산형(diverging) 배색을, 0 이상만 있는 값은 단조 증가하는 순차형(sequential) 배색을 써야 오해 없이 읽힌다. 또 색은 정확한 수치 비교에는 약하므로, 셀 위에 숫자를 함께 표기(annotation)하는 경우가 많다.

## 예시

파이썬에서는 seaborn으로 한 줄이면 그릴 수 있다.

```python
import seaborn as sns
import matplotlib.pyplot as plt

corr = df.corr(numeric_only=True)          # 피처 간 상관관계 행렬
sns.heatmap(corr, annot=True, cmap="coolwarm", center=0)
plt.show()
```

- 진한 빨강 셀 → 강한 양의 상관 (두 피처가 같이 커짐)
- 진한 파랑 셀 → 강한 음의 상관 (하나가 커지면 다른 하나는 작아짐)
- 흰색에 가까운 셀 → 상관 거의 없음
