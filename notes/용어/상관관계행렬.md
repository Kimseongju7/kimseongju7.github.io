## 한 줄 정의

여러 변수(피처) 쌍끼리의 상관계수를 한꺼번에 모아 놓은 정사각 행렬로, 어떤 변수들이 서로 같이 움직이는지를 한눈에 보여준다.

## 자세히

상관계수(보통 피어슨 상관계수)는 두 변수가 얼마나 선형적으로 같이 움직이는지를 -1 ~ +1 사이 값으로 나타낸다. +1에 가까우면 한쪽이 커질 때 다른 쪽도 커지는 강한 양의 관계, -1에 가까우면 반대로 움직이는 강한 음의 관계, 0 근처면 선형 관계가 거의 없다는 뜻이다.

변수가 n개일 때 이 상관계수를 모든 쌍에 대해 계산해 n×n 표로 정리한 것이 상관관계행렬이다. 대각선은 자기 자신과의 상관이므로 항상 1이고, 행렬은 대각선을 기준으로 대칭이다. 숫자만 보면 눈에 잘 안 들어오기 때문에 보통 [[열지도|열지도(heatmap)]]로 색을 입혀 시각화한다 — 진한 색일수록 상관이 강한 식이다.

머신러닝에서는 훈련 전 탐색적 데이터 분석(EDA) 단계에서 즐겨 쓴다. 목표 변수와 상관이 높은 피처를 골라내거나, 서로 상관이 지나치게 높은 피처 쌍(다중공선성)을 찾아 하나를 제거하는 식으로 피처 선택에 활용한다. 히스토그램·상자 플롯이 변수 하나하나의 분포를 보는 도구라면, 상관관계행렬은 변수들 사이의 관계를 보는 도구다.

주의할 점은 상관은 어디까지나 **선형** 관계만 잡아내고, 상관이 높다고 인과관계가 있는 것은 아니라는 것. 곡선형 관계는 상관계수가 0에 가깝게 나올 수 있다.

## 예시

pandas에서는 한 줄로 구한다:

```python
import seaborn as sns

corr = df.corr(numeric_only=True)   # 상관관계행렬 (DataFrame)
sns.heatmap(corr, annot=True, cmap="coolwarm")  # 열지도로 시각화
```

예를 들어 주택 가격 데이터에서 `면적`과 `가격`의 상관이 0.8이면 면적이 큰 집일수록 비싼 경향이 뚜렷하다는 뜻이고, `방 개수`와 `면적`의 상관이 0.95라면 두 피처가 거의 같은 정보를 담고 있으니 하나만 써도 된다는 신호다.
