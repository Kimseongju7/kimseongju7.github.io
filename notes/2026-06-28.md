### 배운 것
- Chirpy(chirpy-starter)는 GitHub Actions 워크플로우가 내장돼 빌드·배포가 자동이라 로컬 Ruby 설치가 필요 없다.
- 옵시디언 그래프는 볼트 내부 전용이고, 발행 스킬이 노트를 Chirpy 포스트(`_posts/`, front matter)로 변환하는 다리 역할을 한다.
- 커밋이 잔디(contribution)에 잡히려면 GitHub 계정에 등록된 이메일로 커밋해야 하며, 이메일을 Private로 둔 경우 `123197704+Kimseongju7@users.noreply.github.com` 형태의 noreply 주소를 쓰면 실제 이메일 노출 없이 연결된다.
- Shell Commands 플러그인은 `{{file_path:absolute}}` 같은 변수를 지원하고, `claude -p "/skill"`로 스킬을 헤드리스 호출할 수 있어 옵시디언 GUI에서 스킬 실행이 가능하다.
- iCloud의 `iCloud~md~obsidian` 컨테이너에 볼트를 두면 옵시디언 모바일과 동기화된다.
