## 요약

깃허브 블로그(Chirpy/Jekyll)를 만들고, 옵시디언으로 글 쓰는 워크플로를 구성했다.

## 한 일

- [[Jekyll·Chirpy|chirpy]] 테마 골랐음 (jekyll 기반)
- 옵시디언 연결해서 글 쓰기로 함 (notes 폴더에 씀)
- 클로드 스킬 5개 만듦: today, format-entry, auto-link, organize-categories, publish
- 잔디 채우는 게 목표, 매일 커밋

## 배운 것

- github pages가 actions로 빌드되면 ruby 로컬에 없어도 됨
- chirpy는 카테고리 2개까지만 됨, 태그는 소문자
