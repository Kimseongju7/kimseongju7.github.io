**Obsidian shell-commands 플러그인이 Claude 세션 없이도 스킬을 트리거하는 방식**
- ✨ 옵시디언에서 실행되는 스킬(tidy-note, publish 등)은 상시 떠있는 클로드 세션이 처리하는 게 아니라, 옵시디언 버튼(Shell Commands 플러그인)이 클릭되는 그 순간 `ECC_GATEGUARD=off claude -p "/tidy-note {{!file_path:absolute}}" --allowedTools "..."` 같은 쉘 명령을 새로 실행하는 구조다. `-p`는 Claude Code CLI를 비대화형(print) 모드로 기동해 현재 대화 세션과 무관한 새 프로세스를 띄우고, `--allowedTools`로 필요한 도구를 미리 승인해 확인 프롬프트 없이 돌아가게 한다. `.obsidian/plugins/obsidian-shellcommands/data.json`에서 각 버튼이 실제로 어떤 명령을 실행하도록 등록돼 있는지 직접 확인할 수 있다.
