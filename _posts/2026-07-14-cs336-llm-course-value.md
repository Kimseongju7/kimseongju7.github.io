스탠퍼드 CS336을 중국어 자막 없이 영어로 끝까지 파고들면 LLM 이해와 영어 실력이 국내(중국) 컴퓨터 전공생 상위 1% 수준이 된다며, 대학 커리큘럼 대신 CS336을 정면돌파하라고 권하는 트윗이다.

## 왜 CS336인가

- "CS336 > 너희 대학 4년 동안의 컴퓨터 수업 99%."
- 중문 자막을 켜지 말고 영어로 빡세게 끝까지 파고들 것. LLM 이해 + 영어 능력이 동시에 올라간다.
- 이 수업의 목표는 딱 하나: **처음부터 끝까지 직접 완전한 LLM 하나를 손으로 만들어 보는 것.**
- 커버 범위: 데이터 전처리 → 모델 훈련 → 최적화·평가 → 배포, 그리고 Transformer, 어텐션 메커니즘, 최신 훈련 방법까지 전체 기술 스택을 한 섹션도 빠짐없이 다룬다.
- 반면 국내(중국) 대학은 아직도 5년 전 PPT로 XP 시스템을 강의하는 수준이라고 꼬집는다.

## 학습 경로 조언

- 끝까지 파고들면 깨닫게 된다: 학교 머신러닝 수업들은 배우면서 검색 좀 하다 보면 저절로 다 수강 완료된 셈이 된다.
- 대학 1학년 신입생은 먼저 CS50으로 기초를 다지고, 그다음 CS336으로 직행하라.
- **선형대수는 반드시 완벽히 익혀라. 미적분보다 100배 중요하다.** 선형대수는 머신러닝의 밑바닥 DNA라서, 이걸 제대로 못 배우면 신경망 이해는 덧셈·뺄셈·곱셈도 모른 채 미분하려는 꼴이다.
