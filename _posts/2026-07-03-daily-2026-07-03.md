## 배운 것

<!-- claude-recall-learn:castleown:start -->
- ✨ 옵시디언 버튼은 Claude Code를 비대화형(headless)으로 실행 → 권한 팝업에 응답할 화면이 없어 파일 쓰기·git 단계에서 멈춤(pending). `settings.local.json`에 허용 규칙을 미리 박아두면 해결
- ✨ `.claude/settings.local.json`은 gitignore 대상이라 커밋·푸시가 안 되는 이 Mac 로컬 전용 파일 (팀 공유용 `settings.json`과 구분)
- ✨ [Jekyll](/posts/jekyll-chirpy/) 빌드 성공 ≠ 배포 성공 — 커밋을 몇 초 차이로 연달아 푸시하면 GitHub Pages 배포가 `deployment_queued`에서 타임아웃날 수 있음
- ✨ `.gitignore`의 넓은 규칙(`data/`)이 의도치 않게 하위 디렉터리(`components/data/`)까지 매칭 → 디스크엔 있지만 커밋 안 되는 파일이 생김. `git add -f`로 개별 강제 추가
- ✨ CommonMark angle-bracket 링크 문법 `[텍스트](<공백 포함 경로>)`으로 URL 인코딩 없이 공백·한글 파일명도 클릭 링크 가능
